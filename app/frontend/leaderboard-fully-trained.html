About Fully-trained Evaluation

Fully-trained models have been fine-tuned or trained on the provided training and validation splits of each target dataset. This represents the traditional supervised learning approach where models learn from labeled examples in the target domain before making predictions on test data.

Evaluation Criteria:

  • Models trained on provided train/validation splits
  • Parameter updates through supervised learning
  • Access to human-labeled examples during training
  • Performance measured on held-out test data
