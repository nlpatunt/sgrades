Submit Your Output File

Upload a CSV or JSON file with your model’s predictions for evaluation on a specific dataset.

Submission Guidelines

  • Download the dataset from the Datasets section
  • Run your model locally and generate predictions in either .csv or .json format
  • The file must include predictions aligned with the dataset’s expected format (e.g., essay_id, predicted_score)
  • Upload the output file along with your name and email
  • Evaluation will be run automatically and results shown on the leaderboard

Example Output File (.csv)

essay_id,predicted_score
essay_001,3.5
essay_002,4.0
...

Submission Details
