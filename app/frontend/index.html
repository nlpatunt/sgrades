⚡ New Requirement: Only complete benchmarks with all 12 datasets appear on the leaderboard!

  • Download all datasets using the "Download All (ZIP)" button above
  • Run your model on all 12 datasets locally
  • Create CSV files for each dataset: ASAP-AES.csv, rice_chem.csv, etc.
  • Create a ZIP file containing all 12 CSV files
  • Upload the ZIP file - evaluation will run across all datasets
