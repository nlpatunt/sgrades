About Zero-shot Evaluation

Zero-shot models generate predictions using only prompts and their pre-trained knowledge, without any training or fine-tuning on the target datasets. This approach tests a model's ability to generalize to new tasks based solely on task descriptions and its existing knowledge.

Evaluation Criteria:

  • No training data or examples from target datasets
  • No parameter updates or fine-tuning
  • Predictions based only on prompts and instructions
  • Performance measured on unseen test data
